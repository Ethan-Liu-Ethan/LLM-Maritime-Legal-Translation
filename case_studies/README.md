# Case Studies

## Overview

This repository contains supplementary data for the qualitative case analysis presented in Sections 4.2.1 and 4.2.2 of the research paper "Evaluating translation performance of large language models with maritime legal translation."

## Contents

Due to space constraints in the main paper, this dataset provides complete contextual information for all terminological and phrasal examples discussed in the case analysis sections.

## File Structure

```
maritime_legal_translation_case_examples/
├── C2E_case_study_context.xlsx
└── E2C_case_study_context.xlsx
```

## Data Description

The dataset consists of two Excel files containing translation cases for both directions:

- **C2E_case_study_context.xlsx**: Complete sentence data for Chinese-to-English translation analysis
- **E2C_case_study_context.xlsx**: Complete sentence data for English-to-Chinese translation analysis

Each Excel file includes:
- Sentence ID numbers
- Source sentences
- Reference translations  
- Translation outputs from all 8 representative LLMs

## Usage

This dataset serves as essential supplementary material for understanding the complete context of translation examples referenced in the qualitative analysis sections of the main research paper.
