# LLM-Maritime-Legal-Translation

Online repository for the paper:

> **Liu, S., & Zhang, Y. (in revision). Evaluating translation performance of large language models in maritime legal translation. *Perspectives: Studies in Translation Theory and Practice*.**

This repository hosts the evaluation dataset, case study materials, appendices, and supplementary notes associated with the above paper. It is maintained to comply with the journal's word-length guidelines while keeping all supporting materials publicly accessible.

---

## Repository Structure


## Contents at a Glance

| Folder | What It Contains | Paper Reference |
|--------|-----------------|-----------------|
| [`dataset/`](dataset/) | 2,000 bilingual sentence pairs from maritime legal texts and the corresponding translations generated by 8 LLMs in both E2C and C2E directions | Section 3.1 |
| [`case_studies/`](case_studies/) | Full source sentences, reference translations, and LLM outputs for all terminological and phrasal examples discussed in the qualitative analysis | Sections 4.2.1 & 4.2.2 |
| [`appendices/`](appendices/) | Lists of international maritime conventions (Appendix 1), Chinese maritime laws and regulations (Appendix 2), and structured translation prompts (Appendix 3) | Section 3.1 |
| [`supplementary_notes/`](supplementary_notes/) | Technical descriptions of evaluation metrics, statistical glossary, figure/table reading guides, and LLM access links | Sections 3.2–3.4 & 4.1 |

Each folder contains its own `README.md` with detailed descriptions of its contents.
